Selected Past Projects and Work

Talk 2020 - [ April 2020 - December 2020 ]

Industry Digital Newspaper
Relevant Context This project aimed to implement a machine-learning-experimental tool that allowed the newspaper readers to search a database of transcripts and find all the US 2020 presidential candidates’ statements on different issues. This project started in early July 2020 and needed to be on time for the first presidential debate by September 29th 2020. It had to be done in collaboration with AWS engineers, since it was sponsored by them, to test one of their newest machine learning products: Kendra
Challenges
  • Pretty tight deadline
  • Recruited and formed a team from scratch
  • At early stages the project was just a concept with a few sketches
  • Synchronize AWS and the newspaper engineering teams making sure to make it work despite the different styles of work and organizational culture
Outcomes
  • Due to the tight deadline, created a very efficient system to onboard the new team members as the team continued building the solution in parallel
  • Coached and helped organize the work and communications across the engineering teams, the product manager and the stakeholders towards the end goal
  • Delivered the product on time and as expected for the first presidential debate
Tools and Techniques
  • Project Management Framework/Method: Kanban Method
  • Capacity Planning
  • Metrics Generation and tracking: Throughput, cycle time, lead time
  • Burn up charts
  • Roadmap
  • Internal Team’s working agreements
  • Project executive status reports
  • Jira - Project Management Tool
  • Google Data Studio for status and metrics reporting
